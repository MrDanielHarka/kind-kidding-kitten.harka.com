Kind Kidding Kitten

The kitten you see in the bottom of the page, can be installed in two ways:
  • Through a browser extension
  • With a JavaScript code

JavaScript Code

  1. Open the source of a website.
  2. Copy the following code to the head section of the code. 
    <script src=
"https://kind-kidding-kitten.harka.com/kkk.js"
defer>
</script>
  3. Save/deploy the site.
  4. Now anybody who visits your website can meet MMM.
